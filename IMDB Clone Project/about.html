    About This Project

    This is an IMDb clone designed to showcase the ability to fetch and display movie data from The Movie Database (TMDb) API. The application features a movie slider, a popular movies section, and the ability to add movies to a favorites list.

    Features

    • Movie Slider showcasing popular movies.
    • Search functionality for finding movies.
    • Dark and light mode toggle.
    • Responsive design for various screen sizes.

    Technologies Used

    The project is built using:

    • HTML
    • CSS
    • JavaScript (ES6)
    • TMDb API

    Future Enhancements

    In future iterations, features such as user authentication, movie reviews, and a personalized favorites list will be added.
